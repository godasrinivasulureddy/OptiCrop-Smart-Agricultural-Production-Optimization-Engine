OptiCrop - Smart Agricultural Production Optimization Engine
🎯 Objective
OptiCrop is an advanced, AI-powered agricultural recommendation and intelligence platform. It acts as a comprehensive decision-support system featuring crop recommendations, real-time weather intelligence, leaf disease diagnosis, SHAP explainable AI, PDF reporting, and a multilingual Farmer AI Assistant. The goal is to optimize agricultural yields by providing precision data and actionable insights directly to farmers.

✨ Features
Crop Suitability Engine: Precision agriculture recommendations based on N, P, K, pH, humidity, temperature, and rainfall.
Vision AI Engine: Leaf image diagnosis to detect diseases and provide natural remedies.
Farmer Assistant: Multilingual advisory chat engine (English, Telugu).
Explainable AI (SHAP): Feature importance breakdowns for why a crop was recommended.
PDF Exporting: Autogenerated, downloadable agronomy reports.
Admin Dashboard: Real-time telemetry, model evaluation metrics, and logs.
💻 Tech Stack
Frontend: React, TypeScript, Tailwind CSS, Vite, Lucide React (Icons).
Backend: Python, FastAPI, SQLAlchemy, JWT Authentication, ReportLab (PDFs).
Database: SQLite (Development), PostgreSQL (Production-ready).
Machine Learning: Scikit-Learn (Random Forest), Joblib.

🧠 ML Models Used
The primary recommendation engine utilizes a Random Forest Classifier trained on historical agronomic data (Nitrogen, Phosphorus, Potassium, Temperature, Humidity, pH, Rainfall, and Season). It was evaluated alongside Logistic Regression, Decision Trees, and KNN, ultimately proving to be the most accurate and robust against overfitting.

🚀 Setup & How to Run
1. Backend Setup
Navigate to the backend and setup environment:
cd backend
python -m venv venv
# Windows: .\venv\Scripts\activate
# Linux/Mac: source venv/bin/activate
Install dependencies:
pip install -r requirements.txt
Configure Environment: Copy .env.example to .env. It defaults to local SQLite.
cp .env.example .env
Seed Initial Data and Train Models: (The database tables are auto-created when the server starts.)
python train_model.py
python seed_admin.py
Start the API Server:
python -m uvicorn app.main:app --reload
2. Frontend Setup
Navigate to the frontend:
cd frontend
npm install
Start the Vite development server:
npm run dev
The application will be accessible at http://localhost:5173.
🔗 Main API Routes
POST /api/auth/login - Authenticate user and receive JWT.
POST /api/auth/register - Create a new user account.
POST /api/predict/recommend - Submit soil data and get ML crop recommendation.
POST /api/vision/analyze - Upload leaf image for disease diagnosis.
GET /api/weather/current - Fetch local weather data.
GET /api/reports/history - Retrieve past predictions/diagnoses.
GET /api/reports/{id}/pdf - Download PDF report.
